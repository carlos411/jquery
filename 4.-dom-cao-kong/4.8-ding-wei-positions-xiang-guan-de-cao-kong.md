# 4.8 取得元素的位置座標

## .offset() 與 .position()

**`.offset()`** 取得相對於頁面的水平方向與垂直方向的距離。

**`.position()`** 取得相對於父元素(往外層找到第一個有設定 position 為非 static 的元素)水平方向與垂直方向的距離。

## .offset() 可以設定位置

註：`.position()` 無法設定位置。

範例：(只能設定 left 及 top)

```javascript
$("#inner_block").offset({ left: 1, top: 1 });
```
