# 5.5 動態事件綁定

html 部份：`5_5_dynamic_binding.html`

```markup
<ul>
  <li><button type="button" class="btn">按鈕</button></li>
</ul>
```

js 部份：

```javascript
$("button.btn").on("click", function(e){

});
```

```javascript
// ul 標籤範圍點擊後，如果是 button.btn 被點擊的話，就觸發
$("ul").on("click", "button.btn", function(e){

});
```
