# 5.2 載入(Load)的相關事件

## DOMContentLoaded

javascript：

```javascript
document.addEventListener("DOMContentLoaded", function(){
  // DOM 載入完成之後
});
```

jQuery：

```javascript
// 寫法一
$(document).ready(function() {
  // DOM 載入完成之後
});

// 寫法二
$(function(){
  // DOM 載入完成之後
});
```



## load

javascript：

```javascript
window.addEventListener("load", function(){
  // 圖片等資源載入完成之後
});
```

jQuery：

```javascript
$(window).on("load", function(){
  // 圖片等資源載入完成之後
});
```



## resize

JavaScript：

```javascript
window.addEventListener("resize", function(){
  // 瀏覽器的寬高有改變時觸發
});
```

jQuery：

```javascript
$(window).on("resize", function(){
  // 瀏覽器的寬高有改變時觸發
});
```
